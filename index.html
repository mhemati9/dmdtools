Welcome

This project provides a collection of tools useful for computing several variants of Dynamic Mode Decomposition. The project includes both python and Matlab scripts, though the python toolset is more complete at the moment. The dmdtools package can be downloaded by click on the icons for .zip or tar.gz above.
